contact_form_app

概要
PHPで作成したシンプルな問い合わせフォームアプリです。
ユーザーからのお問い合わせ内容を受け取り、バリデーションして表示します。
CSRFトークンは送信ごとに新しいものを生成するようにしています。

機能
・お問い合わせ内容の入力フォーム
・入力チェック（バリデーション）
・送信後のサンクスページ表示

使用方法
1. ファイルをサーバーに置く（例: XAMPPのhtdocs）
2. ブラウザで contact.php を開く
3. フォームに入力して送信

ファイル構成
・ContactForm.php：フォーム処理
・contact.php：フォームページ
・thanks.php：送信完了ページ
・validator.php：入力チェック
